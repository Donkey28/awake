## 二、审核风险分析

### 你的 AppID `(已隐藏)` 属于「**社交**」类目高危区

匿名实时聊天 = 微信审核最敏感的类别之一。必须从命名、描述、文案全方位规避以下触雷点：

### 三类雷区

**雷区 A — 社交类目**

| 禁止出现 | 原因 |
|----------|------|
| 聊天/聊天室/聊天广场 | 触发社交类目审核，需要 ICP 许可证 |
| 匹配/遇见/邂逅 | 触发陌生人社交 |
| 匿名聊天/匿名交友 | 直接踩红线 |
| 在线人数/N人在线 | 强调"多人"暗示社交 |
| 发消息/私信/对话 | 社交功能描述 |

**雷区 B — 心理/医疗**

| 禁止出现 | 原因 |
|----------|------|
| 治愈/疗愈/治疗 | 需要医疗资质 |
| 倾诉/倾听/树洞 | 心理咨询暗示 |
| 陪伴/陪你 | 情感服务暗示 |
| 孤独/抑郁/焦虑 | 医学暗示 |

**雷区 C — AI/情感**

| 禁止出现 | 原因 |
|----------|------|
| AI聊天/智能回复 | 当前没有 AI 功能 |
| 情感/情绪/心灵 | 模糊地带，能避则避 |

### 安全定位

**我们是什么**：一个深夜写字的安静角落

**我们不是什么**：
- 不是聊天室（虽然技术上是）
- 不是树洞
- 不是心理咨询
- 不是社交产品

### 当前文案修改建议

| 现有文案 | 审核风险 | 建议 |
|----------|---------|------|
| `还没有人开口。` | 低 | 可保留，「开口」偏中性 |
| `写点什么...` | 低 | 可保留 |
| `今晚还有一些人没睡。` | **中** | 暗示「多人同时在」，改 |
| `还有几个人也醒着。` | **中** | 同上，改 |
| `这个时间，不止你醒着。` | 低 | 可保留 |
| `有一些人和你一起醒着。` | **中** | 改 |
| `睡不着的人，不止你一个。` | 低 | 可保留 |
| `失眠不是惩罚，是独处。` | **中高** | 「失眠」擦边心理，改 |
| `这是夜晚的底部。` | 低 | 可保留 |

### 项目描述（project.config.json description）

当前：`深夜匿名文字空间` — **中风险**，「匿名」+「空间」可能被理解为匿名社交。

建议改为：`深夜写字` 或 `夜晚的空白` — 去「匿名」去「空间」。

## 四、图标方案

**方向**：不用具象图形，用纯色+微弱渐变的抽象表达。

- 背景色：`#0D1117`（深夜蓝黑）或 `#1A1F2E`
- 图形：一根极细横线或微光点，居中
- 参考感觉：凌晨手机屏幕最暗亮度上的一条通知栏细线
- 不做月亮、星星、猫头鹰、路灯 —— 太具象，太社交
- 可以就是一个深蓝底色块 + 微微偏暖的单像素级亮点

简单说：图标就是「暗」本身。
